# 微信读书 Move 快速应用指南

本指南适用于 reMarkable Paper Pro Move 上的“微信读书”应用。界面针对彩色墨水屏设计，翻页或切换界面时可能需要短暂刷新，这是正常现象。

## 一分钟上手

| 想做什么 | 操作方法 |
| --- | --- |
| 打开应用 | 在系统的 AppLoad 页面点“微信读书”图标 |
| 打开一本书 | 在书架点书籍封面，再点“继续阅读” |
| 书架换页 | 在九宫格封面区域向左或向右滑；也可点屏幕左右边缘 |
| 正文翻页 | 用手指在正文左半边或右半边轻点；也可左右滑动 |
| 打开目录 | 从阅读页左边缘向右滑 |
| 关闭目录 | 在目录上横向滑动、点右上角“×”，或点目录外的正文区域 |
| 打开阅读设置 | 从屏幕底部向上滑 |
| 关闭阅读设置 | 在设置面板向下滑 |
| 调节灯光 | 点阅读页右上角分段灯光胶囊 |
| 查看划线评论 | 点正文下方的黑色虚线 |
| 查看注释 | 用笔点红色上标注释编号 |
| 画荧光标记 | 用笔点右侧小圆点，选颜色后在正文上划线 |
| 自由手写 | 展开右侧笔胶囊，先选颜色，再选“写”，直接在页内空白或正文旁书写 |
| 识别一块笔迹 | 用手点目标笔迹块，再点“识别”；不会识别整页 |
| 输入中文 | 在软键盘中选“拼音”或“手写”；候选可点“上页/下页” |
| 配置百度 OCR | 进入“我的 → 百度 OCR → 开启浏览器配置” |
| 返回书架 | 打开阅读设置，点“退出到书架”；目录中也有“回到书架” |
| 退出到系统 | 回到书架，点右上角“退出” |
| 休眠或唤醒 | 短按电源键；合上或打开磁吸保护套也可休眠或唤醒 |

## 1. 打开应用

1. 回到 reMarkable 系统界面。
2. 打开 AppLoad。
3. 点“微信读书”图标。
4. 等待书架出现。墨水屏启动和全屏刷新可能需要几秒，请不要连续重复点击图标。

应用打开后仍会使用设备当前的 Wi-Fi。需要切换网络时，可在底部进入“我的”页面管理 Wi-Fi。

## 2. 第一次登录

如果尚未登录，应用会显示登录二维码：

1. 确保设备已经连接 Wi-Fi。
2. 使用手机微信扫描屏幕上的二维码。
3. 在手机上确认登录。
4. 保持二维码页面打开，等待设备自动进入书架。

二维码失效时，点“重新生成”。不想继续登录时，点“取消”。

以后需要切换账号，可进入底部“我的”，选择“切换微信读书账号”或“退出当前账号”。退出账号不会自动删除设备上已有的书籍封面、阅读文件和其他本地缓存。

## 3. 使用书架

书架每页显示 9 本书。

### 打开书籍详情

点一本书的彩色封面，即可进入书籍详情页。详情页会显示封面、书名、作者、简介和阅读进度。

### 切换下一批书

- 在九宫格封面区域向左滑：下一页书架。
- 在九宫格封面区域向右滑：上一页书架。
- 也可以点屏幕最左侧或最右侧的边缘区域换页。

书架上方会显示当前页数，例如“1 / 3”。书架换页采用直接刷新，不会播放滑动动画。

### 书架顶部操作

- “继续阅读”：直接打开最近阅读的书。
- “下载本页”：下载当前书架页显示的书籍。
- “同步”：刷新微信读书书架和封面。
- “退出”：关闭应用并回到 reMarkable 系统。

### 底部三个页面

- “书架”：查看自己的微信读书书架。
- “发现”：搜索书籍、打开书籍详情，并查看下载记录。
- “我的”：管理 Wi-Fi、账号、登录状态、阅读缓存和微信笔记。

## 4. 书籍详情与下载

进入详情页后：

- 点“继续阅读”：从上次阅读位置打开。若本书尚未缓存，会先下载可立即阅读的开篇章节。
- 点下载按钮：下载整本书；下载过程中可点“取消下载”。
- 点“目录”：先查看章节，再选择要读的章节。
- 点“删除下载”：删除本书的离线阅读文件。
- 点“修复插图”：重新获取或修复本书缺失的图片资源。
- 点左上角“返回”：回到书架。

整本下载完成后，即使暂时没有网络，也可以继续阅读已经缓存的内容。

## 5. 阅读与翻页

### 翻页

请使用手指翻页：

- 轻点正文右半边：下一页。
- 轻点正文左半边：上一页。
- 向左滑：下一页。
- 向右滑：上一页。

笔主要用于选择控件、打开注释和做标记，不会因为笔尖落在正文上而误翻页。

### 页脚

正文底部会保留安全空白，页脚显示：

- 当前页码和总页数。
- 当前阅读进度百分比。

字号、行距、段距或页边距变化后，应用会重新排版；阅读位置会按正文位置恢复，而不是只记住旧页码。

### 返回详情页

从阅读页左上角向下滑，可返回当前书籍的详情页。

## 6. 目录与章节

在阅读页从屏幕左边缘向右滑，即可打开半屏目录。

目录中可以：

- 上下滑动浏览章节。
- 点章节名称直接跳转。
- 点“下载整本”缓存全书。
- 点“回到书架”离开当前书籍。

关闭目录的方法：

- 在目录面板上横向滑动。
- 点目录右上角“×”。
- 点目录之外的正文区域。

## 7. 阅读设置

在阅读页从屏幕底部向上滑，打开“阅读设置”。在设置面板向下滑即可关闭。

可以调整：

- 字号。
- 行距。
- 段距。
- 页边距。
- 首行缩进。
- 字体：微米黑、正黑、默认的霞鹜文楷、思源黑体、思源宋体、寒蝉正楷、寒蝉活宋，以及系统回退字体。
- 字重：“加黑”或“浓黑”。
- 灯光亮度。
- 阅读进度跳转。

常用操作：

- “退出到书架”：保存当前进度并回到书架。
- “重绘本页”：页面残影或局部显示异常时刷新当前页。
- “同步进度”：把当前进度同步到微信读书账号。
- “拉取进度”：读取账号中的云端阅读进度。

阅读设置会自动保存，下次打开应用时继续使用。

## 8. 电量与灯光

阅读页右上角显示电池图标。电池内的黑色格数代表大致剩余电量。

电池旁边是一个分段式灯光胶囊：

- “关”：关闭前灯。
- “25、50、75、100”：设置对应亮度档位。

点选后灯光立即生效。也可以在阅读设置中选择更细的离散亮度档位。

## 9. 划线评论与章节评论

联网阅读时，在当前页停留约 3 秒，应用会尝试加载这一页的微信读书划线评论。快速翻过的页面不会优先加载评论。

- 带评论的文字下方会出现黑色虚线。
- 点虚线可打开评论面板。
- 评论面板约占屏幕大部分区域，可上下浏览评论。
- 点面板外部或关闭按钮可返回正文。

章节末尾如果有章节评论，会显示评论入口，点开后查看本章评论。

评论会保存在本地缓存中。没有网络、当前文字没有评论或微信读书接口暂时不可用时，虚线可能不会出现，但不会影响正常阅读。

## 10. 注释

正文中的注释入口显示为红色上标数字。

1. 用笔点红色上标。
2. 注释会在较大的浮窗中打开。
3. 阅读完后点注释浮窗内的“返回正文”，或点浮窗中的任意空白处关闭。

注释入口优先响应笔，避免用手指翻页时误触。

## 11. 使用荧光笔和自由手写

阅读页右侧会露出一个小圆点，这是笔胶囊入口。颜色选择与工具选择相互独立：
先选颜色不会自动切换荧光笔或自由写，切换工具也不会改变当前颜色。

1. 用笔点右侧小圆点，展开工具栏。
2. 选择绿色、金色、红色或蓝色。
3. 再选择“划”或“写”。

### 荧光笔

选择“划”后，用笔划过正文的一句话或一段话。标记会吸附到实际文字，而不是
保存为漂浮曲线。选择“擦”后，用笔划过已有标记即可清除对应文字标记。

### 自由手写

选择“写”后，可以直接在正文旁或页面空白处书写文字、圈、箭头和符号。笔迹按
原页面坐标保存，不会为了显示“本页笔记”而压缩正文。连续书写时先显示快速黑色
墨迹，停笔后统一刷新为所选颜色。

相邻且时间接近的多笔会组成一个笔迹块。它们不必做 OCR：只想保留原始手写、
纯画线或符号时，写完即可。

### 识别或删除一块笔迹

1. 用手指点目标笔迹块，打开就地操作条。
2. 点“识别”只会把这一块发送给百度 OCR，不会识别整页，也不会打开旧的全屏
   识别窗口。
3. 识别成功后，文字附在该笔迹旁，原始笔迹仍保留。
4. 点“删除”只删除选中的块。

也可以在笔胶囊点“识”，再点要识别的笔迹块。点“清”需要再次确认，确认后才会
清除当前页笔迹；这与删除单个笔迹块不同。

“记”控制重排后是否显示与正文段落关联的手写笔记。无法可靠关联的符号或页内
自由笔迹仍保留在原页面附近。

笔工具只响应笔，手指仍用于翻页和点笔迹操作；笔靠近屏幕时会阻止手掌误触翻页。

## 12. 拼音和手写输入法

点“发现”中的搜索框或其他普通输入框，会打开内置软键盘：

- “拼音”：输入完整拼音，点候选字；候选超过五个时点“上页/下页”。
- “手写”：在白色书写区连续写完，点“识别”，再点百度返回的正确候选。
- “英文”：直接输入字母、数字和符号。密码字段只允许英文模式。

手写模式不会每写两笔就自动识别，必须由用户点“识别”。点“清空”可重新书写。
搜索书籍既可用本地拼音，也可使用手写 OCR 生成的文字。

## 13. 配置百度 OCR

自由手写不需要网络；只有手写输入识别和笔迹块 OCR 需要百度 OCR 与 Wi-Fi。

1. 按[百度 OCR 配置指南](baidu-ocr-configuration-flow.md)在百度智能云创建文字识别
   OCR 应用，并开通手写文字识别。
2. 在应用详情找到 AppID、API Key 和 Secret Key。设备只填写 API Key 与
   Secret Key，AppID 不填。
3. 设备进入“我的 → 百度 OCR → 开启浏览器配置”。
4. 在同一局域网的手机或电脑浏览器中输入设备显示的完整 `https://` 地址。
5. 核对地址后接受本次临时证书提示，输入 6 位配对码、API Key 和 Secret Key。
6. 点“验证并保存”，直到设备显示“百度 OCR 已配置”。

不要把 Key 发给 Agent、写入 `.env` 或放进命令行。配置完成后，凭据只以 `0600`
权限保存在设备私有数据目录中。升级到 2.0.0 时，已有配置会保留；更新百度 OCR 不会清除 DeepSeek，反之亦然。

目前自己的标记和自由笔迹优先保存在设备本地；是否能同步回微信读书账号取决于
后续接口支持。

## 14. 休眠与唤醒

阅读时可以随时：

- 短按一次电源键进入休眠。
- 合上磁吸保护套进入休眠。

休眠画面会显示当前书籍封面。再次短按电源键，或打开磁吸保护套，即可回到阅读位置。

请短按电源键，不要长按；长按属于系统级电源操作。

## 15. Wi-Fi、账号与缓存

在书架底部点“我的”，可以：

- 查看当前 Wi-Fi 状态。
- 扫描、连接或切换已保存的 Wi-Fi。
- 检查微信读书账号状态。
- 扫码登录、切换账号或退出当前账号。
- 续期登录 Cookie。
- 同步微信笔记。
- 开启短时百度 OCR 浏览器配置并查看是否已配置。
- 清理阅读缓存。

“清理阅读缓存”会清除应用生成的书籍展开和排版缓存，不会退出账号，也不会直接删除已经下载的原始整本文件。下次打开相关书籍时，应用会重新生成所需缓存。

## 16. 墨水屏使用提示

- 页面切换后出现一次黑白闪烁，通常是正常的全屏刷新。
- 点过按钮后请等待界面完成刷新，不要连续快速点击。
- 页面出现残影或局部显示不完整时，打开阅读设置并点“重绘本页”。
- 评论、书架同步和首次打开新书都需要 Wi-Fi。
- 下载完成的书可离线阅读；在线评论和云端进度同步仍需要网络。
- 页面自由手写和荧光笔可离线使用；手写 OCR 必须联网并由用户主动点“识别”。
- 如果应用无响应，可短按电源键休眠后再唤醒；仍无响应时回到系统并重新打开应用。

## 17. 推荐的退出顺序

阅读结束时，建议按以下顺序退出：

1. 从底部上滑打开阅读设置。
2. 需要时点“同步进度”。
3. 点“退出到书架”。
4. 若要回到 reMarkable 系统，再点书架右上角“退出”。

这样可以确保阅读位置先保存，再安全返回系统。
